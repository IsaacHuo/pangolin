# 摘 要

过去数年间，伴随着大规模预训练语言模型（Large Language Models, LLM）核心能力的跨越式发展，人工智能的落地形态正自单向对话应用向具备自主推理、环境感知以及多步任务规划执行能力的多智能体系统（Multi-Agent System, MAS）演进。现代智能体（Agent）借由对系统终端甚至第三方网络资源的工具调用（Function Calling）权限，已被大规模布署于办公自动化、代码智能生成与系统运维等真实的高价值生产场景中。然而，当这些具备高权限操作环境及系统访问能力的大模型作为核心决策大脑时，由于语言模型难以天然区分“功能指令数据”与“背景内容数据”的自身模态特性，一旦智能体与不受管控或被恶意操纵的外部业务节点建立数据交换，即刻面临严重的结构外安全隐患。诸如混淆代理攻击（Confused Deputy）、间接提示词注入（Indirect Prompt Injection）、数据投毒与上下文环境篡改等复杂跨模态风险，能够轻易绕过目前基于传统的基于端点签名的实体边界防火墙，使得智能体自身反而成为攻击者的跳板工具，带来极高的数据泄露乃至物理系统破坏威胁。

为了主动揭示并在实地复现、验证、阻断上述隐蔽性极高的新型大模型结构外安全威胁，本课题打破了以往针对此类系统单一的理论攻防设想框架，通过对当前热门的模型跨端通信协议（以 MCP 模型上下文协议等为代表的标准化交互流程）及交互载荷进行深层解析，主导设计并工程化实现了一套全链路、可视化的**大模型智能体安全零信任攻防演练平台（Pangolin 靶场系统）**。该平台不仅能完整在真实环境下复原隐秘的跨节点权限劫持现象，更是进一步作为验证新型安全网关策略理念的高仿真试炼场地，兼具了攻击行为剖解验证与防护组件功能验证的双重测试与演示使命。

在整体架构模式的设计上，该演练靶场采取了安全组件介入中途、强解耦的前后端微服务分离开发范式。
在后端演练控制层，系统核心通过 Python 异步高并发网络框架（FastAPI 与 Uvicorn），基于中间人代理（Man-in-the-Middle, MITM）监听逻辑实现。该服务层可无须改变原智能体业务逻辑代码的条件下，在智能体通信的底层数据套接字（Socket、SSE、Stdio 等流转线路上）进行毫秒级的全流量阻断包获取及还原提取任务。同时平台内嵌搭建了双轨式的审查干预分析模块（Dual-Engine Interception Pipeline）：其第一层防线（L1）搭载了由 Rust 底层性能驱动的 Aho-Corasick 算法与正则表达式集合形成的时空复杂度极优的静态敏感词嗅探阻断器；而负责消解模糊多义及隐藏深意注入欺骗的第二层防线（L2），则创新性地挂载了专职判定指令意图、能够对抗动态变种提示语的高效大模型语义分类器。
在前端全景观测方面，系统构建在最为前沿的 Nuxt 3 （核心依托 Vue 3 组合式 API 及 Vite 底层打包技术）框架之上，结合 WebSocket 和 TypeScript 类型验证建立了持久化的实时大屏总线控制。所有的攻防交互、工具注入过程乃至 L1/L2 两阶段打分放行记录、原始流量报文都可在该控制台之上得以全貌溯源和可视化复盘，彻底扫除了智能体后台隐藏的多重逻辑转包盲区。

本文详细论证并记录了上述智能体攻防演练平台的架构选取、流控拦截中间件深层代码的设计及前端动态图表反馈的技术实现难点突破。依托本平台环境，课题组系统性成功演示与捕获了包含工具欺骗性注册、非法上下文越权投毒、利用历史语境污染误导代理文件清洗的四重典型的渗透演习回路。系统不仅毫无阻断原生通讯的存活心跳反馈地抓包了整个攻击流；且实验统计证明其搭载的 L2 模型驱动拦截验证能在维持毫秒级通讯低损耗判定下有效辨别和遏制多达 99% 的实验沙箱越权渗透注入。本系统的落实验证了零信任双级判定隔离机制在模型与 API 对话防御中运用的技术可操作性，对业界未来大规范建立更加安全的防注入微代理生态防线网格与大模型通信中间件设计提供了兼具研究价值及生产级参照的珍贵平台级依据及演练标尺。
